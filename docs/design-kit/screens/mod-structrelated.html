MOD.STRUCTREL-MODAL · core/modal · large · removeOnClose MOD.STRUCTREL-CARD · o card É o diálogo MOD.STRUCTREL-CLOSE · cor = textcolor da competência (JS) MOD.STRUCTREL-CONTENT · linksclickable/showrelated OFF
O card é o diálogo. A classe local-dimensions-related-modal no root reescreve a casca do core/modal: .modal-header vira display:none, .modal-content fica border:0; background:transparent; box-shadow:none, .modal-body zera o padding, e o .modal-dialog cabe em 620px com radius 24px (o outline de foco do core abraça o card, então o raio casa). O título é passado ao Modal.create, mas o cabeçalho oculto o esconde — o nome vive no card.
As duas travas são de propósito. linksclickable:false deixa os três contadores como número puro (senão um clique abriria o MOD.USAGE por cima deste modal); showrelated:false omite a seção de referenciadas (senão um chip abriria outro MOD.STRUCTREL). É read-only — espiar, não editar (o editor é o MOD.RELATED).
O fechar é o próprio, não o do core. O .btn-close do core está oculto com o cabeçalho, e o restyle global do chip de fechar exclui este modal (:not(.local-dimensions-related-modal)). Este botão é 36×36 branco translúcido no canto, e sua cor de ícone vem do textcolor da competência em JS — para contrastar com o cabeçalho colorido, seja ele qual for.
