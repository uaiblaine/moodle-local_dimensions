Rótulos e ordem são os shipados, casing incluído: Imagem de Fundo primeiro e Imagem do Cartão depois, na ordem em que picture_manager::add_all_filemanagers_to_form() os adiciona (picture_manager.php:163-165). Cada um leva addHelpButton.
O controle shipado é o filemanager do core, não este cartão de uma linha: maxbytes 10 MB, maxfiles 1, subdirs 0, accepted_types web_image (picture_manager.php:100-107). A forma compacta acima — miniatura + nome do arquivo numa linha só — é a to-be: cabe no corpo de um modal, onde o filemanager cheio come altura.
As duas dropzones só existem no modo interno. competency_handler::instance_form_definition() só chama o picture_manager se is_builtin_mode() (competency_handler.php:173-175) — isto é, quando o plugin customfield_picture não está instalado. Com ele instalado, as imagens viram campos personalizados picture e quem desenha a linha é aquele plugin, não este kit (helper.php:470-475).
