Divergências da interpretação anterior (CDS/Anthropic → Moodle DS)
  • Superfícies neutras quentes → grays Bootstrap (frias).
  • Accent único azul → primary (blue-500) + info (cyan) separados, com estados.
  • Hairline 0.5px → 1px (stroke-sm); fonte Anthropic Sans → Noto Sans.
  • Adicionados: fills interativos + estados, foco (WCAG 2.2 AA) e elevação.
  • Nomenclatura flat → semântica --mds-* (primitive→semantic) → port para React do MDS (Moodle 5.3 LTS) vira rename.
  • Movimento: --mds-motion-flash é consumido por JS (WAAPI), não por CSS — é a referência para deduplicar as 10 cópias do flashRow num helper só.
  • Estes tokens são consumidos por zero regras shipadas hoje. Escopo de adoção: superfícies novas ou reformuladas — retro-tokenizar todo o CSS shipado briga com a regra de menos código.
