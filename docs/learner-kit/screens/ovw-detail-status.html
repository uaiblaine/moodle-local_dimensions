About this scale · Skills proficiency

This competency is graded on the institution's four-level skills scale. A learner counts as proficient at Competent or above.

  • Emerging — first attempts; core ideas not yet reliably applied.
  • Developing — applies the skill with guidance and some gaps.
  • Competentproficient — applies the skill independently and reliably.
  • Distinctionproficient — adapts the skill to new, unfamiliar situations.
The modal renders the scale's own description HTML (author-written). The example also lists the levels and marks which count as proficient — that structured list needs the scale items (mdl_scale.scale) + scaleconfiguration; the minimum is just the raw description blob the admin wrote.
